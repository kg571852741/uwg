simulate¶

uwg simulate¶

Commands for simulating UWG models.

uwg simulate [OPTIONS] COMMAND [ARGS]...

model¶

Simulate a UWG model from a JSON model file.

Args:
model_json: Full path to a JSON model file.
epw_path: Full path of the rural .epw file that will be morphed.
uwg simulate model [OPTIONS] MODEL_JSON EPW_PATH

Options

--new-epw-dir <new_epw_dir>¶

Optional argument for the destination directory into which the morphed .epw file is written. The argument passed here will overwrite the new_epw_dir specified in the UWG JSON model file.

--new-epw-name <new_epw_name>¶

Optional argument for The destination file name of the morphed .epw file. The argument passed here will overwrite the new_epw_name specified in the UWG JSON model file.

Arguments

MODEL_JSON¶

Required argument

EPW_PATH¶

Required argument
